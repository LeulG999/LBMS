git\# Library Book Management System (LBMS)



\## Overview



LBMS is a Java application developed using Object-Oriented Programming principles and Maven. The system manages books and book copies and provides operations such as retrieving books, filtering rare books, and sorting books by price.



\## Technologies Used



\* Java 21

\* Maven

\* JUnit 5



\## Project Structure



\* Model Layer

\* Repository Layer

\* Service Layer

\* Presentation Layer



\## Build Instructions



Clone the repository and navigate to the project directory:



```bash

git clone <repository-url>

cd LBMS

```



Build the project:



```bash

mvn clean package

```



\## Run Instructions



Run the generated executable JAR:



```bash

java -jar target/LBMS-1.0-SNAPSHOT.jar

```



\## Testing



Run unit tests:



```bash

mvn test

```
